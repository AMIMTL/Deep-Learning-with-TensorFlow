## Course Overview
This bootcamp teaches deep learning with Google's TensorFlow 2 framework and Keras API. It balances theory with practical implementation, covering everything from neural network basics to GANs and production deployment.

## What You'll Learn
- Use TensorFlow 2.0 and Keras API for deep learning
- Perform image classification with Convolutional Neural Networks (CNNs)
- Apply deep learning to medical imaging
- Forecast time series data with Recurrent Neural Networks (RNNs)
- Generate images with Generative Adversarial Networks (GANs)
- Use deep learning for style transfer
- Generate text with RNNs and NLP
- Serve TensorFlow models through an API
- Use GPUs for accelerated deep learning

## Course Topics Include
- NumPy, Pandas, and Data Visualization crash courses
- Neural network basics and TensorFlow fundamentals
- Keras syntax (Sequential, Functional, and Subclassing APIs)
- Artificial Neural Networks (Densely Connected Networks)
- Convolutional Neural Networks (CNNs)
- Recurrent Neural Networks (RNNs)
- AutoEncoders
- Generative Adversarial Networks (GANs)
- Deploying TensorFlow into production

## Prerequisites
- Python coding skills
- Basic math (derivatives)
